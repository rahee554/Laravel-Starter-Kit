Dark Mode Implementation

All layouts support Bootstrap's native dark mode system using data-bs-theme attribute.

Enable Dark Mode:

<!-- Add to <html> tag -->
<html lang="en" data-bs-theme="dark">

JavaScript Toggle:

function toggleTheme() {
    const html = document.documentElement;
    const currentTheme = html.getAttribute('data-bs-theme');
    const newTheme = currentTheme === 'dark' ? 'light' : 'dark';
    html.setAttribute('data-bs-theme', newTheme);
    localStorage.setItem('theme', newTheme);
}

// Load saved theme
const savedTheme = localStorage.getItem('theme') || 'light';
document.documentElement.setAttribute('data-bs-theme', savedTheme);

System Preference Detection:

// Detect system preference
if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
    document.documentElement.setAttribute('data-bs-theme', 'dark');
}
üí° CSS Variables: All layouts use CSS custom properties that automatically switch based on the theme:
  • --auth-bg, --auth-text, --auth-card-bg
  • --admin-bg, --admin-sidebar-bg, --admin-text
  • Colors transition smoothly when theme changes

Bootstrap Integration

All layouts are built with Bootstrap 5.3.8 following best practices.

Architecture:

SCSS Structure:

‚îî‚îÄ‚îÄ Custom Components

Custom styles OVERRIDE Bootstrap using !important flags.

Form Components:

All form controls are completely overridden while maintaining Bootstrap's semantic HTML structure:

  • .form-control - Custom input styling with focus states
  • .form-select - Custom dropdown styling
  • .form-check-input - Custom checkbox/radio with ::before pseudo-elements
  • .btn-primary - Custom gradient buttons

Admin Responsive Sidebar:

The admin sidebar layout uses Bootstrap's JavaScript for responsive behavior:

  • Desktop: Fixed sidebar with toggle collapse
  • Mobile: Sidebar collapses completely, triggered by hamburger menu
  • Pure JavaScript toggle (no jQuery required)
  • Smooth transitions with CSS
// Sidebar toggle implementation
document.getElementById('sidebarToggle')?.addEventListener('click', function() {
    document.querySelector('.admin-layout-sidebar')
            .classList.toggle('sidebar-collapsed');
});

Customization Guide

Change Default Layout:

// In .env file
STARTERKIT_AUTH_LAYOUT=particles
STARTERKIT_ADMIN_LAYOUT=sidebar

// In config/starterkit.php
'auth_layout' => env('STARTERKIT_AUTH_LAYOUT', 'particles'),
'admin_layout' => env('STARTERKIT_ADMIN_LAYOUT', 'sidebar'),

Customize Colors:

// In resources/css/auth/_variables.scss
$primary: #00aaff;  // Your brand color
$body-bg: #f8f9fa;  // Light mode background

// In resources/css/auth/_variables.scss (dark mode)
$body-bg-dark: #0f172a;  // Dark mode background

Add Custom Layout:

  1. Create new Blade file: resources/views/layouts/starterkit/auth/my-layout.blade.php
  2. Extend base structure and use auth components
  3. Add SCSS in resources/css/auth/_layouts.scss
  4. Build assets: npm run build
